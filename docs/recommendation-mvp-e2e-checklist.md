# Recommendation MVP E2E Checklist

## Preconditions
- [ ] `.env`에 `EXPO_PUBLIC_OPENAI_API_KEY`가 설정되어 있다.
- [ ] 테스트 계정으로 로그인 가능하다.
- [ ] 옷장에 최소 1개 이상의 업로드 이미지가 있다.
- [ ] 네트워크 연결 상태가 정상이다.

## Core Success Flow
- [ ] 업로드한 아이템에서 추천 화면으로 진입한다.
- [ ] `오늘 원하는 스타일`, `상황`, `날씨`, `예산`을 입력/선택한다.
- [ ] `AI 추천 받기`를 2~3번 연속 탭해도 중복 요청이 발생하지 않는다.
- [ ] 로딩 오버레이가 표시되고, 완료 후 사라진다.
- [ ] 추천 결과에 `전체 코디 요약`, `스타일 팁`, 카테고리별 아이템이 노출된다.
- [ ] 네이버/쿠팡/무신사 버튼 탭 시 외부 링크가 정상적으로 열린다.
- [ ] 좋아요/별로예요 탭 시 성공 알림이 보이고, 에러 없이 종료된다.

## History Flow
- [ ] 추천 생성 후 히스토리 화면 새로고침 시 최신 항목이 상단에 보인다.
- [ ] 히스토리 카드에 스타일/상황/날씨/예산/피드백이 표시된다.
- [ ] 히스토리 카드에 추천 요약이 표시된다.
- [ ] 히스토리 카드에 추천 아이템 미리보기(최대 3개)가 표시된다.

## Failure Cases
- [ ] OpenAI API 키를 제거한 상태에서 추천 요청 시 안내 메시지가 표시된다.
- [ ] 네트워크를 끊은 상태에서 추천 요청 시 실패 알림과 `다시 시도` 버튼이 표시된다.
- [ ] 잘못된 링크(또는 열 수 없는 링크) 상황에서 링크 오류 알림이 표시된다.
- [ ] 히스토리의 `recommendation` 파싱 실패 데이터가 있어도 화면이 깨지지 않는다.

## Regression Check
- [ ] 로그인/회원가입/업로드/홈/마이페이지 기존 플로우에 오류가 없다.
- [ ] 추천 화면에서 스타일 칩 선택 및 입력 필드 동작이 기존과 동일하다.
- [ ] 앱 크래시 없이 추천 요청 반복 테스트를 3회 이상 통과한다.
